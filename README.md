# Borsa izleme sitesi

> https://tr.investing.com/ Sitesinden alınan websocketi React kullanarak kullanıcıya sunan bir site.

# Kullanılan Paketler
- React
- Chart.js
- react-use-websocket

# Site Özellikleri
- Artış/düşüş durumuna göre değişen kart rengi
- Grafik gösterimi
- Tablo sayesinde son 20 veriyi görmek
